LIGA DEPORTIVA

Este proyecto cuenta con una página principal con iconos superiores que direccionan a webs de cada club. Detalles y ubicación de estadios mediante el uso de un navegador más añadido de noticias. Sección de galerias destacadas y videos exclusivos más el apartado de equipos ganadores. Uso de Json para ver equipos más ganadores. Se hace uso de formularios y generador de presupuesto ante una posible compra.
